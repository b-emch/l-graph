Que se passe-t-il quand un mineur est héritier dans une succession ?

Guide: Succession après décès

Un mineur peut-il hériter ?

Oui, les mineurs peuvent hériter, mais ils sont considérés comme "légalement incapables" de prendre des décisions indépendantes. Un parent survivant ou un administrateur légal doit les aider à accepter ou refuser l'héritage.

Autorité décisionnelle

Le parent survivant ou l'administrateur désigné agit au nom de l'enfant, requis d'agir dans son meilleur intérêt. Le tribunal peut nommer un tuteur temporaire neutre si des conflits surviennent entre le parent et l'enfant.

Options d'héritage

Les mineurs ont les mêmes choix que les adultes :

  • Refuser la succession (surtout si elle a des dettes)
  • L'accepter inconditionnellement
  • L'accepter "avec bénéfice d'inventaire" (protection contre les dettes cachées)

Toutes les décisions nécessitent l'approbation du tribunal.

Gestion des actifs après l'héritage

Le parent survivant gagne le pouvoir de gestion sur les actifs hérités tout en maintenant les droits parentaux. Il peut jouir du bien (comme vivre dans une maison héritée) et doit prendre des décisions bénéficiant à l'enfant. L'argent est placé sur un compte bloqué accessible quand le mineur atteint 18 ans.

Cas particulier : Parents divorcés

Si le parent décédé était divorcé, le parent survivant conserve la responsabilité de la gestion malgré le divorce. Les parents peuvent prévenir cela par testaments spécifiant les restrictions sur les droits du ex-partenaire sur les actifs hérités, bien que l'autorité parentale reste intacte.
